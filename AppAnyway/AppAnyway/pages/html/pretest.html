性能测试  
   测试方法

  从唯品会取测试数据，其中唯品会品牌页取10个数据，商品页每页10个数据。将AppAnyWay与唯品会进行性能比较，测试过程中每步记录内存使用情况。

   说明

  测试1~3 每次需要退出app，在继续进行下一测试。

    测试1
  1. 进入app，获取唯品会品牌页
  2. 点击某品牌进入商品页
  3. 滑动3页
  4. 回退到品牌页
  5. 重复2~4步3次，每次选择不同品牌
    测试2
  1. 进入app，获取唯品会品牌页
  2. 点击某品牌进入商品页
  3. 滑动3页
  4. 回退到品牌页
  5. 重复2~4步3次，每次相同品牌
    测试3
  1. 进入app，获取唯品会品牌页
  2. 点击某品牌进入商品页
  3. 滑动5页
  4. 回退到品牌页
