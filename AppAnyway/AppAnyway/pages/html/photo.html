<照片    

类名：AWPhoto

概述:

    拍照、选择照片。

方法:

  • choosePhoto

    功能：

        从相册中选择照片。

    方法定义： 
    iOS：
    -(void) choosePhoto:( NSDictionary *param )
    Android：
    public AWResult choosePhoto (String arguments)
     参数（Parameters）：

    allowsEditing:
      类型：bool
      描述：图片是否裁剪
    width:
      类型：float
      描述：图片最大宽度
    height：
      类型：float
      描述：图片最大高度

    回调函数（Callback）：

    result：
      类型：字符串
      描述：返回图片在TMP中的URI
    error：
      类型：字符串
      可能错误代码有：0005、0407、1302

    例子： 
    AW.invoke("AWPhoto", "choosePhoto", param,
    function(data) {
      if (data.success) {
        $("#page_photo .content .choose").attr('src', data.result);
      } else {
        $("#page_photo .content .choosePhoto").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

    备注:

    注： sdk编译版本需Android 4.4以上

    测试: 选照片：
