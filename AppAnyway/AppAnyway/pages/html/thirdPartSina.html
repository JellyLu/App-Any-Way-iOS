<Sina    

类名：AWThirdPartSina

概述:

    Sina微博登录、分享到微博。

方法:

  • login

    功能：

        Sina微博登录。

    方法定义： 
    iOS：
    -(void) login:( NSDictionary *param )
    Android：
    public AWResult login (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：JSON字符串
      描述：返回userID与凭证，如：
    {
     "userID":"1749108241",
     "accessToken":
     "2.00PxE4uBb37qLBbd38a930b3Mm278C"
    }
    error：
      类型：字符串
      可能错误代码有：1701、1702、1703、1704、1705、1706、0001

    例子： 
    AW.invoke("AWThirdPartSina", "login", {},
    function(data) {
      if (data.success) {
        $("#page_thirdPartSina .content .login").html("userID: " + data.result.userID + " accessToken:" + data.result.accessToken);
      } else {
        $("#page_thirdPartSina .content .login").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

    测试： 登录：

  • shareMsg

    功能：

        分享内容到Sina微博。

    方法定义： 
    iOS：
    -(void) shareMsg:( NSDictionary *param )
    Android：
    public AWResult shareMsg (String arguments)
     参数（Parameters）：

    url：
      类型：字符串
      描述：链接
    imageUrl：
      类型：字符串
      描述：图片的URL
    title：
      类型：字符串
      描述：标题
    description：
      类型：字符串
      描述：内容

    回调函数（Callback）：

    result：
      类型：bool
      描述：返回是否分享成功
    error：
      类型：字符串
      可能错误代码有：1701、1702、1703、1704、1705、1706、0001

    例子： 
    var param = {
      "url": "http://www.appanyway.com",
      "imageUrl": "http://www.appanyway.com/anywayWeb/awdownload/test2.png",
      "title": "AppAnyWay",
      "description": "AppAnyWay是由绵阳恒达信息科技有限公司和Web智能技术实验室联合研发的手机应用开发中间件，是一款基于Android、iOS原生+Html5混合开发中间件，开发者无需掌握复杂的移动开发技术，只需具备Html(5)开发能力，即可快速开发出一款移动App。"
    };
    AW.invoke("AWThirdPartSina", "shareMsg", param,
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_thirdPartSina .content .shareMsg").html('成功');
        } else {
          $("#page_thirdPartSina .content .shareMsg").html('成功');
        }
      } else {
        $("#page_thirdPartSina .content .shareMsg").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
