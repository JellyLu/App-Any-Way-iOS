<文件系统    

类名：AWFileSystem

概述:

    保存、删除、读取缓存文件，获取缓存文件/文件夹大小,获取文件夹下某类文件名列表，iOS所有文件放在沙盒中Library/Caches下，通过folderName和fileName指定要处理的文件或者文件夹；Android所有文件放在文件数据区data/data/packageName(你的应用包名)/files下，通过folderName和fileName指定要处理的文件或者文件夹。

方法:

  • saveFile

    功能：

        所给内容保存到指定文件夹下的文件中。

    方法定义： 
    iOS：
    -(void)saveFile:( NSDictionary *param )
    Android：
    public AWResult saveFile (String arguments)
     参数（Parameters）：

    content：
      类型：字符串
      描述：需要保存的数据
    folderName：
      类型：字符串
      描述：文件夹/子路
    fileName：
      类型：字符串
      描述：文件名

    回调函数（Callback）：

    result：
      类型：bool
      描述：是否保存成功
    error：
      类型：字符串
      可能错误代码有：0003、0406、0407

    例子： 
    AW.invoke("AWFileSystem", "saveFile", {
      "folderName": "cache",
      "fileName": "test1.doc",
      "content": "123"
    },
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_fileSystem .content .saveFile").html('保存成功');
        } else {
          $("#page_fileSystem .content .saveFile").html('保存失败');
        }
      } else {
        $("#page_fileSystem .content .saveFile").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

  • getFileList

    功能：

        获取路径下文件夹中某类型文件的文件名。

    方法定义： 
    iOS：
    -(void) getFileList:( NSDictionary *param )
    Android：
    public AWResult getFileList (String arguments)
     参数（Parameters）：

    folderPath：
      类型：字符串
      描述：文件夹路径
    type：
      类型：字符串
      描述：文件类型，如doc、txt、mp3、mp4等等，如果type为空字符串，则返回文件夹中所有文件的文件名。

    回调函数（Callback）：

    result：
      类型：数组
      描述：文件名列表
    error：
      类型：字符串
      可能错误代码有：0003

    例子： 
    AW.invoke("AWFileSystem", "getFileList", {
      "folderName": "cache",
      "type": ""
    },
    function(data) {
      if (data.success) {
        var html = "";
        for (var i = 0, len = data.result.length; i < len; ++i) {
          html += " " + data.result[i] + "\n";
        }
        $("#page_fileSystem .content .getFileList").html(html);
      } else {
        $("#page_fileSystem .content .getFileList").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
