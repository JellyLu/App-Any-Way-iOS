<QQ    

类名：AWThirdPartQQ

概述:

    QQ登录、分享到QQ好友与QQ空间。

方法:

  • login

    功能：

        QQ登录。

    方法定义： 
    iOS：
    -(void) login:( NSDictionary *param )
    Android：
    public AWResult login (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：JSON字符串
      描述：返回昵称与凭证，如：
    {
      "userName":"Jelly",
      "accessToken":
      "FF9BB90883899B1E7733190C6362A473"
    }
    error：
      类型：字符串
      可能错误代码有：1606、1607、1608、1609、1610、1611、1612、0001

    例子： 
    AW.invoke("AWThirdPartQQ", "login", {},
    function(data) {
      if (data.success) {
        $("#page_thirdPartQQ .content .login").html("userName: " + data.result.userName + " accessToken:" + data.result.accessToken);
      } else {
        $("#page_thirdPartQQ .content .login").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

    测试： 登录：

  • shareMsg

    功能：

        分享内容到QQ好友或QQ空间。

    方法定义： 
    iOS：
    -(void) shareMsg:( NSDictionary *param )
    Android：
    public AWResult shareMsg (String arguments)
     参数（Parameters）：

    type：
      类型：字符串
      描述：分享到QQ好友、QQ空间，”qq”为QQ好友，”qzone”为QQ空间。
    url：
      类型：字符串
      描述：链接
    imageUrl：
      类型：字符串
      描述：图片的URL
    title：
      类型：字符串
      描述：标题
    description：
      类型：字符串
      描述：内容

    回调函数（Callback）：

    result：
      类型：bool
      描述：返回是否分享成功
    error：
      类型：字符串
      可能错误代码有：1601、1602、1603、1604、1605、0001

    例子： 
    var param = {
      "type": "qzone",
      "url": "http://www.appanyway.com",
      "imageUrl": "http://www.appanyway.com/anywayWeb/awdownload/test2.png",
      "title": "AppAnyWay",
      "description": "AppAnyWay是由绵阳恒达信息科技有限公司和Web智能技术实验室联合研发的手机应用开发中间件，是一款基于Android、iOS原生+Html5混合开发中间件，开发者无需掌握复杂的移动开发技术，只需具备Html(5)开发能力，即可快速开发出一款移动App。"
    };
    AW.invoke("AWThirdPartQQ", "shareMsg", param,
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_thirdPartQQ .content .shareMsgQzone").html('成功');
        } else {
          $("#page_thirdPartQQ .content .shareMsgQzone").html('成功');
        }
      } else {
        $("#page_thirdPartQQ .content .shareMsgQzone").html(data.error.code + ":" + data.error.desc);
      }
    });
