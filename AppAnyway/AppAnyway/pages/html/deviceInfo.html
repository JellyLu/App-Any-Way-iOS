<设备信息    

类名：AWDeviceInfo

概述:

    获取设备相关信息，屏幕宽度、高度、设备名称、imei、系统版本号。

方法:

  • getBodyWidth

    功能：

        获取设备屏幕宽度。

    方法定义： 
    iOS：
    -(void) getBodyWidth:( NSDictionary *param )
    Android：
    public AWResult getBodyWidth (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：字符串
      描述：屏幕宽度，如320
    error：
      类型：字符串
      可能错误代码有：0301

    例子： 
    AW.invoke("AWDeviceInfo", "getBodyWidth", {},
    function(data) {
      if (data.success) {
        $("#page_deviceInfo .content .bodyWidth").html(data.result);
      } else {
        $("#page_deviceInfo .content .bodyWidth").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

    测试: 宽度：

  • getImei

    功能：

        设备唯一32位标识。

    方法定义： 
    iOS：
    -(void) getImei:( NSDictionary *param )
    Android：
    public AWResult getImei (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：字符串
      描述：设备唯一32位标识,如：
    BE553E30F4954CF38483098A089C8271(iOS)、863908027430839（Android）
    error：
      类型：字符串
      可能错误代码有：0304

    例子： 
    AW.invoke("AWDeviceInfo", "getImei", {},
    function(data) {
      if (data.success) {
        $("#page_deviceInfo .content .imei").html(data.result);
      } else {
        $("#page_deviceInfo .content .imei").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

    补充说明：

    iOS中由于设备获取设备IMEI是私有类，不能通过App Store审核，所以采用设备UUID替代。
