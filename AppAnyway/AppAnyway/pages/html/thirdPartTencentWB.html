<腾讯微博    

类名：AWThirdPartTencentWB

概述:

    腾讯微博登录、分享到腾讯微博。(注： 由于腾讯微博SDK不再更新，已不支持armv64，所以仅能在不支持armv64的工程中可以使用。)

方法:

  • login

    功能：

        腾讯微博登录。

    方法定义： 
    Android：
    public AWResult login (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result:
      类型：JSON字符串
      描述：返回userID与凭证，如：
    {
      "userName":"Jelly",
      "accessToken":
    "49096194af449500a838712ca5954665",
      "openId":
    "3CB1AFC5BF0EB32BDD9F57306C6C023B"
    }
    error：
      类型：字符串
      可能错误代码有：1801、1802

    例子： 
    AW.invoke("AWThirdPartTencentWB", "login", {},
    function(data) {
      if (data.success) {
        $("#page_thirdPartTencentWB .content .login").html("userName: " + data.result.userName + " accessToken:" + data.result.accessToken + " openId:" + data.result.openId);
      } else {
        $("#page_thirdPartTencentWB .content .login").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    Android：Android 4.0 及以上可用

    测试: 登录：由于腾讯微博SDK不再更新，已不支持armv64，所以仅能在不支持armv64的工程中可以使用。

  • shareMsg

    功能：

        分享内容到腾讯微博。

    方法定义： 
    Android：
    public AWResult shareMsg (String arguments)
     参数（Parameters）：

    url：
      类型：字符串
      描述：链接
    imageUrl：
      类型：字符串
      描述：图片的URL
    title：
      类型：字符串
      描述：标题
    description：
      类型：字符串
      描述：内容

    回调函数（Callback）：

    result:
      类型：bool
      描述：返回是否分享成功
    error：
      类型：字符串
      可能错误代码有：1801、1802

    例子： 
    var param = {
      "url": "http://www.appanyway.com",
      "imageUrl": "http://www.appanyway.com/anywayWeb/awdownload/test2.png",
      "title": "AppAnyWay",
      "description": "AppAnyWay是由绵阳恒达信息科技有限公司和Web智能技术实验室联合研发的手机应用开发中间件，是一款基于Android、iOS原生+Html5混合开发中间件，开发者无需掌握复杂的移动开发技术，只需具备Html(5)开发能力，即可快速开发出一款移动App。"
    };
    AW.invoke("AWThirdPartTencentWB", "shareMsg", param,
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_thirdPartTencentWB .content .shareMsg").html('成功');
        } else {
          $("#page_thirdPartTencentWB .content .shareMsg").html('成功');
        }
      } else {
        $("#page_thirdPartTencentWB .content .shareMsg").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    Android：Android 4.0 及以上可用

    测试： 登录：由于腾讯微博SDK不再更新，已不支持armv64，所以仅能在不支持armv64的工程中可以使用。
