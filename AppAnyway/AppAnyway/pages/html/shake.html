<摇一摇    

类名：AWShake

概述:

    开启和关闭摇一摇，当开启摇一摇时，检测到设备正在摇动震动。

方法:

  • shake

    功能：

        开启或关闭摇一摇。

    方法定义： 
    iOS：
    -(void) shake:( NSDictionary *param )
    Android：
    public AWResult shake (String arguments)
     参数（Parameters）：

    isShake：
      类型： bool
      描述：是否开启摇一摇，true为开启，false为关闭。

    回调函数（Callback）：

    result：
      类型：bool
      描述：是否正在摇
    error：无

    例子： 
    AW.invoke("AWShake", "setShake", {
      "isShake": true
    },
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_shake .content .setShakeOn").html("成功");
        } else {
          $("#page_shake .content .setShakeOn").html("失败");
        }
      } else {
        $("#page_shake .content .setShakeOn").html(data.error.code + ":" + data.error.desc);
      }
    });
    AW.invoke("AWShake", "setShake", {
      "isShake": false
    },
    function(data) {
      if (data.success) {
          if (data.result) {
          $("#page_shake .content .setShakeOff").html("成功");
        } else {
          $("#page_shake .content .setShakeOff").html("失败");
        }
      } else {
        $("#page_shake .content .setShakeOff").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
