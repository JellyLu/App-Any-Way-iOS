<网络    

类名：AWNetWork

概述:

    判断是否连接到网络，获取当前连接网络类型。

方法:

  • isConnectNetWork

    功能：

        判断是否连接到网络。

    方法定义： 
    iOS：
    -(void) isConnectNetWork:( NSDictionary *param )
    Android：
    public AWResult isConnectNetWork (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：bool
      描述：返回是否连接到网络
    error：无

    例子： 
    isConnectNetWork: function() {
    AW.invoke("AWNetWork", "isConnectNetWork", {},
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_netWork .content .isConnectNetWork").html("已联网");
        } else {
          $("#page_netWork .content .isConnectNetWork").html("未联网");
        }
      } else {
        $("#page_netWork .content .isConnectNetWork").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

  • getSSID

    功能：

        获取SSID(Service Set IDentifier)，即当前连接Wi-Fi的公开名称。

    方法定义： 
    iOS：
    -(void) getSSID:( NSDictionary *param )
    Android：
    public AWResult getSSID (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：字符串
      描述：如果连接到Wi-Fi，返回Wi-Fi名称，否则返回空字符串。
    error：
      类型：字符串
      描述：可能错误代码有：1201

    例子： 
    AW.invoke("AWNetWork", "getSSID", {},
    function(data) {
      if (data.success) {
        $("#page_netWork .content .getSSID").html(data.result);
      } else {
        $("#page_netWork .content .getSSID").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

  • getNetWorkType

    功能：

        获取当前连接网络类型。

    方法定义： 
    iOS：
    -(void) getNetWorkType:( NSDictionary *param )
    Android：
    public AWResult getNetWorkType (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：字符串
      描述：如果未连接到网络，返回“没有网络连接”，连接到网络返回，”WIFI”、”WWAN(4G)”、”WWAN(3G)”、”WWAN(2G)”。
    error：
      类型：字符串
      描述：可能错误代码有：1201

    例子： 
    AW.invoke("AWNetWork", "getNetWorkType", {},
    function(data) {
      if (data.success) {
        $("#page_netWork .content .getNetWorkType").html(data.result);
      } else {
        $("#page_netWork .content .getNetWorkType").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
