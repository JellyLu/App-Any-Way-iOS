<支付宝    

类名：AWAlipay

概述:

    支付宝快捷支付。

方法:

  • payOrder

    功能：

        支付宝快捷支付。

    方法定义： 
    iOS：
    -(void) payOrder:( NSDictionary *param )
    Android：
    public AWResult payOrder (String arguments)
     参数（Parameters）：

    tradeNO：
      类型：字符串
      描述：订单编号
    notifyURL：
      类型：字符串
      描述：回调URL
    productName：
      类型：字符串
      描述：商品名称
    price：
      类型：字符串
      描述：价格
    productDescription：
      类型：字符串
      描述：商品描述

    回调函数（Callback）：

    result：
      类型：字符串
      描述：付款成功结果
    error：
      类型：字符串
      可能错误代码有：1401、1402、1403、1404、1405、0001、0003

    例子： 
    var param = {
      "tradeNO": "201504190001",
      "notifyURL": "http://www.appaynway.com/anywayWeb/mobi_cbasync.action",
      "productName": "chanel",
      "price": "0.01",
      "productDescription": "coco chanel"
    };
    AW.invoke("AWAlipay", "payOrder", param,
    function(data) {
      if (data.success) {
        $("#page_alipay .content .payOrder").html(data.result);
      } else {
        $("#page_alipay .content .payOrder").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

    测试: 付款：由于账号原因，暂不提供本功能测试
