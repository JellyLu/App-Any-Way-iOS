<联系人    

类名：AWLinkMan

概述:

    通讯录，添加、删除联系人、查看所有联系人。

方法:

  • addLinkMan

    功能：

        添加联系人。

    方法定义： 
    iOS：
    -(void) addLinkMan:( NSDictionary *param )
    Android：
    -public AWResult addLinkMan (String arguments)
     参数（Parameters）：

    nickName：
      类型：字符串
      描述：昵称
    firstName：
      类型：字符串
      描述：姓
    lastName：
      类型：字符串
      描述：名
    tel：
      类型：字符串
      描述：电话号码

    回调函数（Callback）：

    result：
      类型：bool
      描述：是否添加成功
    error：无

    例子： 
    AW.invoke("AWLinkMan", "addLinkMan", {
      "nickName": "aa",
      "firstName": "a",
      "lastName": "b",
      "tel": "15881616205"
    },
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_linkMan .content .addLinkMan").html('成功');
        } else {
          $("#page_linkMan .content .addLinkMan").html('失败');
        }
      } else {
        $("#page_linkMan .content .addLinkMan").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

    测试： 添加联系人：

  • allLinkMan

    功能：

        获取所有联系人的姓名与号码。

    方法定义： 
    iOS：
    -(void) allLinkMan:( NSDictionary *param )
    Android：
    public AWResult allLinkMan (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：JSON字符串
      描述：返回所有联系人的电话号码与姓名，如：
    {
     {
      "realName":"Jelly",
      "tel":"15881616205"
     },
     {  "realName":"John",
      "tel":"15881616206"
     }
    }
    error：
      类型：字符串
      可能错误代码有0004、0901

    例子： 
    AW.invoke("AWLinkMan", "allLinkMan", {},
    function(data) {
      if (data.success) {
        var ob = data.result;
        var html = "";
        for (var i = 0, count = ob.length; i < count; i++) {
          html += " " + ob[i].realName + ":" + ob[i].tel + ";";
        }
        $("#page_linkMan .content .allLinkMan").html(html);
      } else {
        $("#page_linkMan .content .allLinkMan").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

  • deleteLinkMan

    功能：

        删除指定联系人。

    方法定义： 
    iOS：
    -(void) deleteLinkMan:( NSDictionary *param )
    Android：
     public AWResult deleteLinkMan (String arguments)
     参数（Parameters）：

    fullName：
      类型：字符串
      描述：姓名（由firstName和lastName组成的）。
    tel：
      类型：字符串
      描述：电话号码

    回调函数（Callback）：

    result：
      类型：bool
      描述：是否删除成功
    error：无

    例子： 
    AW.invoke("AWLinkMan", "deleteLinkMan", {
      "fullName": "ab",
      "tel": "15881616205"
    },
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_linkMan .content .deleteLinkMan").html('成功');
        } else {
          $("#page_linkMan .content .deleteLinkMan").html('失败');
        }
      } else {
        $("#page_linkMan .content .deleteLinkMan").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
