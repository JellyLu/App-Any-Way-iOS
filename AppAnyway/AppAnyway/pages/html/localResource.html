<常用功能    

类名：AWLocalResource

概述:

    系统的一些本地功能，如拨打电话、判断设备是否越狱、判断设备是否安装某个APP、通过浏览器打开一个链接、通过包名打开一个APP。

方法:

  • call

    功能：

        拨打电话。

    方法定义： 
    iOS：
    -(void) call:( NSDictionary *param )
    Android：
    public AWResult call (String arguments)
     参数（Parameters）：

    number：
      类型：字符串
      描述：电话号码

    回调函数（Callback）：

    result:
      类型：bool
      描述：是否成功调用拨号功能
    error：
      类型：字符串
      可能错误代码有：0003、0501

    例子： 
    AW.invoke("AWLocalResource", "call", {
      "number": "15881616204"
    },
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_localResource .info .call").html('成功');
        } else {
          $("#page_localResource .info .call").html('失败');
        }
      } else {
        $("#page_localResource .info .call").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
