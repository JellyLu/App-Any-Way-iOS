<百度导航    

类名：AWBaiduNavigation

概述:

    百度导航。

方法:

  • startNavigation

    功能：

        开始导航。

    方法定义： 
    iOS：
    -(void) startNavigation:( NSDictionary *param )
    Android：
    public AWResult startNavigation (String arguments)
     参数（Parameters）：

    startX：
      类型：double
      描述： 起点坐标经度
    startY：
      类型：double
      描述：起点左边纬度
    sType：
      类型：int
      描述：起点坐标类型，0为原始的经纬度坐标，1为从百度地图中获取的左边。
    endX：
      类型：double
      描述： 终点坐标经度
    endY：
      类型：double
      描述：终点左边纬度
    eType：
      类型：int
      描述：终点坐标类型，0为原始的经纬度坐标，1为从百度地图中获取的左边。

    回调函数（Callback）：

    result：
      类型：bool
      描述：是否成功进行导航
    error：
      类型：字符串
      可能错误代码有：0003、1501、1502、1503、0006、1504、1505、1506

    例子： 
    var param = {
      "startX": "116.307854",
      "startY": "40.055878",
      "sType": "0",
      "endX": "116.403875",
      "endY": "39.915168",
      "eType": "0"
    };
    AW.invoke("AWBaiduNavigation", "startNavigation", param,
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_baiduNavigation .content .navigation").html('成功');
        } else {
          $("#page_baiduNavigation .content .navigation").html('失败');
        }
      } else {
        $("#page_baiduNavigation .content .navigation").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0及以上可用，注：在Android 5.0.0及以上不可用
