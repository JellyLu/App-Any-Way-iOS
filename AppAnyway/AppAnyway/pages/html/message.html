<短信    

类名：AWMessage

概述:

    短信。

方法:

  • shareMsg

    功能：

        发送短信。

    方法定义： 
    iOS：
    -(void) shareMsg:( NSDictionary *param )
    Android：
    public AWResult shareMsg (String arguments)
     参数（Parameters）：

    content：
      类型：字符串
      描述：短信内容

    回调函数（Callback）：

    result：
      类型：bool
      描述：短信分享是否成功
    error：
      类型：字符串
      可能错误代码有1101、1102、1103

    例子： 
    AW.invoke("AWMessage", "shareMsg", {
      "content": "AppAnyWay是由绵阳恒达信息科技有限公司和Web智能技术实验室联合研发的手机应用开发中间件，是一款基于Android、iOS原生+Html5混合开发中间件，开发者无需掌握复杂的移动开发技术，只需具备Html(5)开发能力，即可快速开发出一款移动App。"
    },
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_message .content .shareMsg").html('分享成功');
        } else {
          $("#page_message .content .shareMsg").html('分享失败');
        }
      } else {
        $("#page_message .content .shareMsg").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
