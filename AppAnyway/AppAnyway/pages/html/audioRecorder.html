<音频    

类名：AWAudioRecorder

概述:

    录制音频，录制、暂停、停止。

方法:

  • recordOrPause

    功能：

        播放或暂停背景音乐。

    方法定义： 
    iOS：
    -(void) recordOrPause:( NSDictionary *param )
    Android：
    public AWResult recordOrPause (String arguments)
    参数（Parameters）：

    rate：
      类型：float
      描述：采样率

    回调函数（Callback）：

    result：
      类型：字符串
      描述：录音/暂停
    error：
      类型：字符串
      可能错误代码有：0701、0702

    例子： 
    AW.invoke("AWAudioRecorder", "recordOrPause", 
    {
      "rate": "22050.0f"
    },
    function(data) {
      if (data.success) {
        $("#page_audioRecorder .content .recordOrPause").html(data.result);
      } else {
        $("#page_audioRecorder .content .recordOrPause").html(data.error.code + ":" + data.error.desc);
      }
    });
    可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

    测试： 开始/暂停:

  • stop

    概述：

    停止录音

    方法定义： 
    iOS：
    -(void) stop:( NSDictionary *param )
    Android：
     public AWResult stop (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：字符串
      描述：音频本地URI
    error：
      类型：字符串
      可能错误代码有0702

    例子： 
    AW.invoke("AWAudioRecorder", "stop", {},
    function(data) {
      if (data.success) {
        $("#page_audioRecorder .content .stop").html(data.result);
      }else {
        $("#page_audioRecorder .content .stop").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
