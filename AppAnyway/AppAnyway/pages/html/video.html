<视频    

类名：AWVideo

概述:

    录制视频、选择视频。

方法:

  • chooseVideo

    功能：

        从相册中选择视频。

    方法定义： 
    iOS：
    -(void) chooseVideo:( NSDictionary *param )
    Android：
    public AWResult chooseVideo (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：字符串
      描述：返回视频在设备中的URI
    error：
      类型：字符串
      可能错误代码有0005、0407、1302

    例子： 
    AW.invoke("AWVideo", "chooseVideo", {},
    function(data) {
      if (data.success) {
        $("#page_video .content .choose").attr('src', data.result);
      } else {
        $("#page_video .content .chooseVideo").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

    测试: 选择视频：
