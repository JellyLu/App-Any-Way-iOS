<APP信息    

类名：AWAppInfo

概述:

    获取APP相关信息，包名、版本号、更新软件、退出软件。

方法:

  • getPackageName

    功能：

        获取包名。

    方法定义： 
    iOS：
    -(void) getPackageName:( NSDictionary *param )
    Android：
    public AWResult getPackageName(String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result:
      类型：字符串
      描述：APP包名，如com.app.anyway
    error：
      类型：字符串
      可能错误代码有：0202

    例子： 
    AW.invoke("AWAppInfo", "getPackageName", {},
    function(data) { 
      if (data.success) {
        $("#page_appInfo .content .packageName").html(data.result);
      } else {
        $("#page_appInfo .content .packageName").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用

    测试: 包名：

  • updateSoft

    功能：

        更新软件。

    方法定义： 
    iOS：
    -(void) updateSoft:( NSDictionary *param )
    Android：
    public AWResult updateSoft(String arguments)
     参数（Parameters）：

    url：
      类型：字符串
      描述：更新软件的地址
      注：并不检测是否更新，所以更新地址必须为可下载的APK文件。

    回调函数（Callback）：

    result：
      类型：字符串
      描述：是否成功跳转到App Store更新软件。
    error：
      类型：字符串
      可能错误代码有：0003、0201

    例子： 
    AW.invoke("AWAppInfo", "updateSoft", {"url":"http://www.appanyway.com/awdownload/demo/1.0/AnyWay.apk"},
    function(data) {
      if (data.success) {
        $("#page_appInfo .content .updateSoft").html(data.result);
      } else {
        $("#page_appInfo .content .updateSoft").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：由于App Store审核不希望App内有更新软件的提示，所以iOS删除该功能。

    Android：Android 4.0 及以上可用

    测试： 更新软件：由于App Store审核不希望App内有更新软件的提示，所以iOS删除该功能，不提供测试
