类名：AWScan

概述:

    扫描二维码、条形码。

方法:

  • scan

    功能：

        扫描二维码或条形码。

    方法定义： 
    iOS：
    -(void) scan:( NSDictionary *param )
    Android：
    public AWResult scan (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：字符串
      描述：扫描结果
    error：
      类型：字符串
      可能错误代码有1301、0007

    例子： 
    AW.invoke("AWScan", "scan", {},
    function(data) {
      if (data.success) {
        $("#page_scan .content .scan").html(data.result);
      } else {
        $("#page_scan .content .scan").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
