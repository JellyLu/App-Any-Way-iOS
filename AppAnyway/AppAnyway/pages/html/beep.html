<蜂鸣    

类名：AWBeep

概述:

    蜂鸣。

方法:

  • beep

    功能：

        蜂鸣。

    方法定义： 
    iOS：
    -(void) beep:( NSDictionary *param )
    Android：
    public AWResult beep (String arguments)
     参数（Parameters）：

    soundID：
      类型：int
      描述：系统声音的ID
    numberOfLoops：
      类型：int
      描述：蜂鸣循环次数

    回调函数（Callback）：

    result：
      类型：bool
      描述：蜂鸣是否成功
    error：
      类型：字符串
      可能错误代码有0801

    例子： 
    AW.invoke("AWBeep", "beep", {
      "soundID": "1001",
      "numberOfLoops": "2"
    },
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_beep .content .beep").html('成功');
        } else {
          $("#page_beep .content .beep").html('失败');
        }
      } else {
        $("#page_beep .content .beep").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
