<震动    

类名：AWVibrae

概述:

    震动。

方法:

  • vibrate

    功能：

        震动。

    方法定义： 
    iOS：
    -(void) vibrate:( NSDictionary *param )
    Android：
    public AWResult vibrate (String arguments)
     参数（Parameters）：

        无

    回调函数（Callback）：

    result：
      类型：bool
      描述：震动是否成功
    error：无

    例子： 
    AW.invoke("AWVibrate", "vibrate", {},
    function(data) {
      if (data.success) {
        if (data.result) {
          $("#page_vibrate .content .vibrate").html('成功');
        } else {
          $("#page_vibrate .content .vibrate").html('失败');
        }
      } else {
        $("#page_vibrate .content .vibrate").html(data.error.code + ":" + data.error.desc);
      }
    });
     可用性:

    iOS：iOS 6.0 及以上可用
    Android：Android 4.0 及以上可用
